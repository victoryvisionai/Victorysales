Business Model

Victory Vision AI is a multi-tenant SaaS platform that provides AI-powered sales and marketing automation to small and mid-size businesses. Customers subscribe to one of three tiers (Bronze $99/mo, Silver $450/mo, Gold $1,250/mo) and use the platform to manage their outreach, campaigns, and paid advertising from a single workspace.

The Google Ads integration is available on the Gold plan. Customers connect their own Google Ads accounts to Victory Vision using OAuth 2.0 — Victory Vision never stores or shares Google account credentials. Each customer retains full ownership of their Google Ads account. Victory Vision does not resell advertising inventory or take a percentage of ad spend.

Tool Access / Use

The tool is used by individual business owners and their teams who are subscribed to the Gold plan. After connecting their Google Ads account via OAuth, users can:

  • View campaign performance dashboards (impressions, clicks, conversions, cost) inside Victory Vision
  • Create and manage Search, Display, Performance Max, and YouTube campaigns
  • Receive AI-generated ad copy and optimization recommendations
  • Apply budget adjustments and bid changes from within the platform
  • Pause or activate campaigns and ad groups
  • Pull attribution data to match Google Ads clicks to CRM contacts in the platform

All API calls are made server-side via Victory Vision's n8n Cloud automation layer using the authenticated user's OAuth access token. End users access the tool directly through the web application at victoryvision.app.

Tool Design

Authentication: Users initiate OAuth 2.0 authorization from the Settings page of the Victory Vision web app. After authorizing, the access token and refresh token are stored encrypted in Victory Vision's Supabase database and associated with the user's account. Tokens are refreshed automatically before expiry.

Data flow: When a user views their Ads dashboard or triggers an optimization, Victory Vision's backend makes API calls to Google Ads using that user's token with the developer token passed in the header. Results are stored in Victory Vision's database and rendered in the user-facing dashboard. No Google Ads data is shared between customers.

Automation: Victory Vision's AI layer (powered by Claude / Anthropic) can generate campaign recommendations and ad copy. Users review recommendations before any changes are applied to their Google Ads account — no fully autonomous changes are made without user confirmation.

API Services Called

  • GoogleAdsService — Search queries for reporting: campaign performance, ad group metrics, keyword stats, conversion data
  • CampaignService — Create, update, pause, and activate campaigns
  • AdGroupService — Create and manage ad groups within campaigns
  • AdGroupAdService — Create and manage ads (responsive search ads, display ads) within ad groups
  • CampaignBudgetService — Read and update campaign budgets
  • CustomerService — Retrieve account information and accessible customer list for the authorized user
  • ConversionActionService — Read conversion actions to map ad clicks to CRM contacts
